# @weflocc/kata Package (private)

## Installation

```
yarn add @weflocc/kata
```

## Updating

```
yarn upgrade @weflocc/kata@latest
```


## Configuration

The plugin can be configured through `<your-studio-folder>/config/kata.json`:

```json
{
  "add-config": "here"
}
```
